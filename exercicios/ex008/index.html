Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântico).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto Grande & Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Com a tag BIG e a tag SMALL, entretanto a tag big se tornou obsoleta na nova atualização do HTML5 e a tag Small continua sendo usada.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado

Usando a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Usando a tag INS.

Texto Sobrescrito

Para inserir coisas do tipo x20+3

Foi utilizado a tag SUP, que serve para fazer com que os números que fiquem entre a tag, fique um pouco a cima da posição original de um texto comum, servindo como um "elevado a".

Texto Subscrito

Para inserir coisas do tipo H2O

Foi utilizado a tag SUB, que serve para fazer com que os números que fiquem entre a tag, fique um pouco abaixo da posição original de um texto comum, normalmente usado nas fórmulas de física.
